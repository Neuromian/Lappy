- 🤖 完全由 Trae AI编写，展示了 AI 辅助开发的未来
- 🎯 基于 Flutter 构建的跨平台桌面应用

## 技术栈

- **Flutter**: 强大的跨平台开发框架
- **fluent_ui**: 现代化的 Windows UI 设计语言
- **GetX**: 高性能的状态管理解决方案
- **flutter_acrylic**: 窗口特效支持
- **shared_preferences**: 本地数据持久化
